# TING

AI WhatsApp assistant SaaS for Indian real-estate agents.

Live: https://convorian.in

## Stack

- Next.js 14 + React + TypeScript
- Supabase Postgres
- Groq primary LLM with GLM fallback
- Meta Cloud API for WhatsApp
- Razorpay for payments
- Vercel for hosting

## Documentation

Full project documentation lives in [docs/TING_SOURCE_OF_TRUTH.md](docs/TING_SOURCE_OF_TRUTH.md).
